# Boogi On And On

## 부기온앤온 소개
- 코로나 시대로 인한 언택트 대학생활로 서로에 대해 잘 모르는 대학친구들에 대해 알 수 있는 실명제 SNS 앱
- 한성대학교 컴퓨터 공학과 2022 캡스톤 디자인 출품 작품

## 주요 기능 소개

<table>
<tr>
  <th style = "width: 25%;"> 커뮤니티를 생성하고 게시글 및 댓글을 작성할 수 있습니다.</th>

  <th style = "width: 25%;"> 프로필에서 내가 작성한 글과 댓글을 확인할 수 있습니다. </th>
</tr>
